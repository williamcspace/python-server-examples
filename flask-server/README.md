# Flask Server

install flask
```shell
> pip install virtualenv
```

```shell
> mkdir myproject
> cd myproject
> virtualenv venv
```

activate environment
```shell
> mkdir myproject
> cd myproject
> virtualenv venv
```

```
> . venv/bin/activate
```

deactivate
```
> deactivate
```

```
> pip install Flask
```


```shell
> export FLASK_APP=hello-world.py
> flask run
```

or 

```shell
> export FLASK_APP=hello-world.py
> python -m flask run
```
